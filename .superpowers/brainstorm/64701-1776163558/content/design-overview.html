Key Decisions

  • 3 new files — InterestObjects (canvas), DiscoveryCounter (DOM), InterestCard (DOM)
  • 1 new data file — interests.js with all 9 items, positions, and descriptions
  • State management — React context or lifted state in SpaceExperience for discovered items
  • Persistence — localStorage saves which interests have been found across visits
  • Performance — Sprites are GPU-cheap; no new post-processing; textures loaded once
