Sprite Visuals

Idle State
  • Emoji rendered to canvas texture (crisp at any zoom)
  • Soft golden glow aura (additive blending)
  • Gentle vertical bob (sin wave, ~0.3 units)
  • Slow rotation on Y axis
  • Scale: small enough to feel like space debris, large enough to notice
Hover / Near State
  • Glow intensifies (brighter aura)
  • Scale up slightly (1.0 → 1.2)
  • Cursor changes to pointer
  • Subtle particle sparkle around object
Discovered State
  • Glow shifts from gold to the section's accent color
  • Stays visible but dimmer (already found)
  • Click still shows the card (re-readable)
Click → Discovery
  • Brief burst animation (particles fly outward)
  • Glass card appears in DOM overlay
  • Counter pulses and increments
  • Card shows: emoji + name + achievement line
